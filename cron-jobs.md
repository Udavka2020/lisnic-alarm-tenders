## ⚙️ GitHub Actions (дублирование)

Единственная задача, продублированная на GitHub:

- Ищет тендеры на achizitii.md по CPV кодам

- Работает даже если сервер упал

Файл: `.github/workflows/tender-monitor.yml`

## 🛠️ Восстановление после сбоя

При восстановлении сервера с нуля:

1. Склонировать репозиторий: `git clone https://github.com/Udavka2020/lisnic-alarm-tenders.git`
2. Восстановить Hermes из бэкапа (backup-full.sh)
3. Все cron задачи восстановятся автоматически (хранятся в `~/.hermes/cron/`)
4. Проверить: `cronjob list` (через Hermes)

```bash
# Если cron не восстановился — пересоздать ключевые задачи:
cronjob create --schedule "5 8,12,16 * * 1-5" --skill tenders-moldova --prompt "Поиск тендеров..."
cronjob create --schedule "0 21 * * *" --skill home-finance --prompt "Ежедневный отчёт..."
# Полный список — в этом файле
```

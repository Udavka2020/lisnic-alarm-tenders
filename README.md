# Lisnic Alarm — Тендеры

Управление тендерными заявками, документацией и автоматизация Lisnic Alarm S.R.L.

## Структура

```
├── tenders/           # Тендерные документы (по номеру тендера)
│   └── 21656191/      # Пример: Panoramic MAI
├── templates/         # Шаблоны документов (OMF 115, etc.)
├── scripts/           # Скрипты автоматизации
├── company/           # Профиль компании, сертификаты
└── .github/workflows/ # GitHub Actions
```

## Автоматизация

- Поиск тендеров по CPV ключевым словам
- Генерация DOCX с таблицами по формам OMF 115
- Проверка дедлайнов и уведомления

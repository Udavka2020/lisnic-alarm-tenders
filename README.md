## GitHub Actions

**Tender Monitor** — автоматический поиск тендеров:

- **Расписание:** Пн–Пт, 8:00 и 14:00 (Chișinău)
- **Поиск:** по CPV кодам на achizitii.md
- **Результат:** создаёт issue с найденными тендерами
- **Ручной запуск:** Actions → Tender Monitor → Run workflow
